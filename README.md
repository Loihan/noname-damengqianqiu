# 大梦千秋扩展包

## 扩展包简介

大梦千秋是一个为无名杀游戏设计的扩展包，包含6个精心设计的武将角色，每个角色都拥有独特的技能机制和专属技能语音效果，部分角色拥有专属出牌音效。

- **作者**: Loihan
- **版本**: 3.3
- **环境推荐**:无名杀诗笺版、千幻聆音扩展包 
- **说明**: 本扩展包用于自娱自乐，不涉及平衡性调整。武将强度超标，与原游戏平衡性不符。

## 安装与目录结构
请将解压后的文件夹放置于 noname/extension/ 目录下。
```text
大梦千秋/
├── README.md
├── LICENSE
├── extension.js
├── info.json
├── character/          # 武将模块目录
│   ├── sgz_*.js        # 三国志系列武将代码
│   ├── shj_*.js        # 山海经系列武将代码
│   └── mfsd_*.js        # 魔法时代系列武将代码
├── image/              # 武将原画
└── audio/              # 音频资源目录
    └── [武将ID]/
        ├── die/      # 阵亡配音
        ├── skill/      # 技能配音
        └── cards/      # 劫持后的卡牌配音
```

## 技术特性
- **模块化架构**：扩展采用 ES6 模块化标准。每一位武将的逻辑、翻译、台词及品质均封装在 `character/` 文件夹下的独立`js` 文件中。`extension.js` 仅作为核心逻辑调度与数据合并入口。
- **自动品质设置**: 所有武将自动设置为传说品质
- **音频劫持系统**: 智能识别并替换特定武将的卡牌语音
