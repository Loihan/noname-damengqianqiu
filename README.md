# 大梦千秋扩展包

## 扩展包简介

大梦千秋是一个为无名杀游戏设计的扩展包，包含6个精心设计的武将角色，每个角色都拥有独特的技能机制和专属技能语音效果，部分角色拥有专属出牌音效。

- **作者**: Loihan
- **版本**: 1.1
- **说明**: 本扩展包用于自娱自乐，不涉及平衡性调整。武将强度超标，与原游戏平衡性不符。

## 安装与目录结构
请将解压后的文件夹放置于 noname/extension/ 目录下。
```text
大梦千秋/
├── README.md
├── LICENSE
├── extension.js        # 扩展主程序
├── info.json
├── character/          # 独立武将模块
├── image/              # 武将原画
└── audio/              # 音频资源
    └── [武将ID]/
```

## 包含武将

### 1. 梦姜维 - 炽剑补天
- **体力**: 4/9
- **技能**: 九伐
- **衍生技能**：薪燃，逐日，绝烬，孤炬，幽明
- **特色**: 拥有多个衍生技能，随着游戏进程逐步解锁

### 2. 梦诸葛诞
- **体力**: 4/9
- **技能**: 定叛，伐逆，焚玉，困诛，孤绝
- **特色**: 互扣体力上限的爆将

### 3. 梦钟会
- **体力**: 1
- **技能**: 权计，忿肆，觊天，逐月，兴伐
- **特色**: 拥有专属出牌音效。手牌恒为标记个数，受伤延迟结算，直伤扣体力上限

### 4. 梦赵云
- **体力**: 1/2
- **技能**: 绝境，龙魂，救主
- **特色**: 龙魂冲阵，可提升的体力上限，多牌龙魂的特殊效果

### 5. 梦黄月英
- **体力**: 4/6
- **技能**: 集巧，璇和，长明，绮梦
- **特色**: 使用锦囊根据花色带特殊效果，特殊观星，大过牌

### 6. 梦郭嘉
- **体力**: 3
- **技能**: 观虚，窥天，极慧，天殇
- **特色**: 拥有专属出牌音效。动态体力上限，动态二版星阵，虚空印牌

## 技术特性
- **模块化设计**: 每个武将独立成模块，便于维护和扩展
- **自动品质设置**: 所有武将自动设置为传说品质
- **音频劫持系统**: 智能识别并替换特定武将的卡牌语音
